# Bundled fixes.csv — data sources & licenses

`fixes.csv` is a merged enroute-fix dataset built by `scripts/build_fixes.py`.

| source tag | provider | license |
|---|---|---|
| `faa-nasr` | FAA NASR subscription | US public domain |
| `openaip`  | OpenAIP reporting points | CC BY-NC-SA 4.0 — attribution + ShareAlike, **non-commercial only** |
| `ofmx`     | open flightmaps designated points | OFMA General Users' License — attribution required |

Because the bundle includes OpenAIP data, the dataset as a whole is **CC BY-NC-SA**:
non-commercial use only, with attribution, and any redistribution must keep the
same license.
